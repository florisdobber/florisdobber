Soccer Analysis

Using R I scraped player data for all players in the Eredivisie from transfermarkt.com. Following this I converted it to a CSV and then used Tableau to illustrate the squad age profile of each team in the Eredivisie. This project was inspired by visuals I had seen from Opta and other soccer stats fanatics.
